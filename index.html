Project 1: Ouderavond website

Beschrijving: Deel 1 van de avond gaat elk koppel uitleg geven aan de wederzijdse ouders over de organisatie van de opleiding. Alle onderwerpen die je in jouw website moet opnemen gaan in de projecturen aan jou gepresenteerd worden. Elk koppel heeft maximaal 45 minuten de tijd om alle verplichte onderwerpen te behandelen.

Project 2: JavaScript game

Beschrijving: Je gaat in drietallen een website opleveren. De website moet aan de onderstaande eisen te voldoen:

  • Een homepage met het principe van "Donate to play / play to donate"
  • Per student een pagina met diens favoriete goede doel
  • Per student een pagina met de persoonlijke motivatie waarom je voor jouw goede doel hebt gekozen
  • Per student een portfolio pagina waarin je jezelf presenteert voor een mogelijke aanbieder van een stageplaats

  • Per student een pagina waarin je het doel en de werking van jouw eigen Javascript game beschrijft
  • Een menu dat op alle pagina's exact hetzelfde is
